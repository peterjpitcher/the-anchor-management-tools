# Project Standards

This directory contains the single source of truth for all development standards for the Anchor Management Tools project.

## Index
-   [**UI & UX Standards**](./UI_UX.md): Component usage, styling, layouts.
-   [**Database Standards**](./DATABASE.md): Schema, RLS, Naming.
-   [**Code Quality**](./CODE_QUALITY.md): TypeScript, Formatting, Naming.
-   [**Architecture**](./ARCHITECTURE.md): Patterns, Structure, Data Flow.
-   [**Process**](./PROCESS.md): Git, Testing, Deployment.

## Usage for AI Agents
When reviewing code or planning changes, consult these files first.
1.  **Check**: Does the code violate any rules in the relevant standard file?
2.  **Enforce**: Suggest changes that align with these standards.
3.  **Verify**: Use the "Review Checklist" at the bottom of each standard file.
