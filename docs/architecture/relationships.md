## Auth Flow

```
Request
  └── middleware.ts
        ├── Public path? → pass through
        ├── Short link host? → pass through
        └── Authenticated path + no session? → redirect /auth/login
              │
              └── (authenticated)/layout.tsx
                    ├── supabase.auth.getUser() — hard check
                    ├── getUserPermissions() — RBAC load
                    ├── No permissions? → redirect /portal/shifts
                    └── Render AuthenticatedLayout (permissions in context)
                          │
                          └── Page / Server Action
                                ├── checkUserPermission(module, action) — per-feature RBAC
                                └── createClient().auth.getUser() — re-verify in actions

Cron routes:  authorizeCronRequest() — Bearer CRON_SECRET
Webhook routes: Twilio/PayPal signature validation
Guest routes: token lookup via src/lib/guest/tokens.ts
```
