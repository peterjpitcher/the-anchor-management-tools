## Auth Model

1. **Middleware** (`src/middleware.ts`) — active, refreshes Supabase session, redirects unauthenticated users from non-public paths
2. **Layout guard** (`src/app/(authenticated)/layout.tsx`) — double-checks `auth.getUser()` and loads RBAC permissions; redirects staff-only users to `/portal/shifts`
3. **RBAC** — `checkUserPermission(module, action, userId)` in `src/services/permission.ts`; roles: `super_admin`, `manager`, `staff`
4. **Cron auth** — `authorizeCronRequest()` in `src/lib/cron-auth.ts` checks `Authorization: Bearer CRON_SECRET`
5. **Guest auth** — short-lived URL tokens in `src/lib/guest/tokens.ts`
6. **API key auth** — `src/app/(authenticated)/settings/api-keys/actions.ts`
