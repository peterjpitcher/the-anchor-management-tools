Build a `'use client'` component that renders a coloured banner strip showing clock-in status.

Props:
```typescript
interface FohClockBandProps {
```

Behavior:
1. On mount, call `getOpenSessions()` from `@/app/actions/timeclock` and check if any session matches the given `employeeId`.
2. If clocked in: show a green-tinted banner with text "Clocked in since HH:MM" (format the `clock_in_at` timestamp using `formatTime12Hour` from `@/lib/dateUtils`) and a "Clock Out" button (ds/ Button, variant="secondary", size="sm").
3. If not clocked in: show a neutral/gray banner with text "Not clocked in" and a "Clock In" button (ds/ Button, variant="primary", size="sm").
4. Clock In button calls `clockIn(employeeId)` server action, then refreshes state.
5. Clock Out button calls `clockOut(employeeId)` server action, then refreshes state.
6. Use `useState` for loading state during clock actions. Disable buttons while loading.
7. Use `useEffect` on mount for initial fetch. Use `useCallback` for action handlers.
8. Show a subtle `Spinner` from `@/ds` while initial status is loading.
